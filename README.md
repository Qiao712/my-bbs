### 论坛系统
* 贴子发表: 
  * 支持富文本，可插入图片
  * 贴子热度排序
  * 贴子全文搜索
* 评论回复功能: 支持两级评论
* 点赞功能: 贴子评论点赞
* 消息通知
* 用户私信
* 后台管理:
  * 贴子、用户、评论、板块管理
  * RBAC 权限管理
* 统一的文件管理，过期图片定期删除

### 重点考虑/待优化的点：
* 消息推送、站内消息系统
* 实时聊天
* 点赞、浏览量、热榜优化
* 接口缓存、热点数据缓存
* 关注者动态推送
